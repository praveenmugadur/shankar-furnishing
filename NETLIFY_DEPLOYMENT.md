# 🚀 Netlify Deployment Guide

## Quick 2-Minute Deployment

### Method 1: Drag & Drop (No Account Needed)

1. **Open this link in your browser:**
   ```
   https://app.netlify.com/drop
   ```

2. **Open File Explorer** (Windows Key + E) and navigate to:
   ```
   c:\Users\pmugadur\OneDrive - RadiSys Corporation\Documents\Website_Dev
   ```

3. **Select ALL files** in the Website_Dev folder:
   - Click inside the folder
   - Press `Ctrl + A` (selects all)
   - OR manually select: `index.html`, `style.css`, `script.js`, and `images` folder

4. **Drag and drop** all selected files onto the Netlify Drop page
   - You'll see a upload progress indicator
   - Wait 20-30 seconds

5. **Your site is LIVE!** 🎉
   - You'll get a URL like: `https://random-name-123456.netlify.app`
   - This URL works **immediately** and is accessible from **any device**

---

### Customize Your URL (Optional)

1. After deployment, you'll see a "Claim this site" button
2. Sign up for a free Netlify account (takes 30 seconds)
3. Click on your site name at the top
4. Go to **Site settings** → **Site details** → **Change site name**
5. Enter: `shankar-furnishing`
6. Your new URL: `https://shankar-furnishing.netlify.app` ✨

---

### Update Your Website Later

**To update prices or change content:**

1. Edit your local files (`script.js` for prices)
2. In Netlify dashboard, click **"Deploys"** tab
3. Drag the updated `Website_Dev` folder to the deploy area
4. Site updates in seconds!
